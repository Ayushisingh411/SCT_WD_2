# ⏱️ Premium Stopwatch Web Application

A modern, interactive, and feature-rich Stopwatch Web Application built using **HTML5, CSS3, and Vanilla JavaScript**. Designed with a premium UI inspired by Apple, Linear, and Framer, this project combines high-precision timing with elegant animations and productivity-focused features.

![CSS](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)

## ✨ Features

- ⏱️ High-precision stopwatch with milliseconds
- ▶️ Start, Pause, Resume, and Reset controls
- 🏁 Unlimited lap recording
- 📊 Live statistics
  - Total Time
  - Total Laps
  - Fastest Lap
  - Slowest Lap
  - Average Lap
- 📤 Export lap data in CSV, TXT, and JSON formats
- 🌙 Dark / Light Theme
- 🎨 Multiple accent color themes
- ⌨️ Keyboard shortcuts
- 🔊 Synthesized sound effects using Web Audio API
- 🎉 Achievement badges and confetti animations
- 📈 Session analytics and productivity dashboard
- 💾 LocalStorage support for saving preferences and session history
- 📱 Fully responsive design
- ✨ Smooth animations and glassmorphism UI

## ⌨️ Keyboard Shortcuts

| Key | Action |
|------|--------|
| Space | Start / Pause |
| L | Add Lap |
| R | Reset Stopwatch |
| T | Toggle Theme |
| F | Fullscreen Mode |

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Ayushisingh411/SCT_WD_2.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No installation or build process is required.
